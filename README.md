### 🛒 ShopEase – Modern E-commerce Web Application
ShopEase is a modern, scalable e-commerce web application built using a cutting-edge frontend stack. It demonstrates best practices in UI development, state management, API handling, and performance optimization.

## 🚀 Live Demo
  https://shopease-dk.netlify.app/

## 📂 Project Structure
  src/
  │
  ├── components/     # Reusable UI components
  ├── pages/          # Application pages
  ├── hooks/          # Custom React hooks
  ├── utils/          # Helper functions
  ├── assets/         # Images & static files
  └── App.tsx         # Main app entry

## ⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/Sreelatha-Vallabhaneni/ShopEase.git
cd ShopEase
2. Install dependencies
npm install
3. Run development server
npm run dev

## App will be available at:
http://localhost:8080/

## 🧪 Running Tests
npm run test

## 📦 Build for Production
npm run build
Preview build:
npm run preview
